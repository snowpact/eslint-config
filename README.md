# Snowpact eslint config

## Installation

```shell script
yarn add @snowpact/eslint-config -D
```

## Usage


### React
Using `.eslintrc.json`:
```json
{
  "extends": "@snowpact/eslint-config/react"
}
```

Using `.eslintrc.js`:
```javascript
module.exports = {
  extends: '@snowpact/eslint-config/react'
}
```
